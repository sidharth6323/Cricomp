# stocks-app


This is a SPA(Single Page Application) using React.js to display live stock data obtained from ws://stocks.mnet.website.

### Prerequisites

```
NodeJs or npm
```

### Installing and Running
```
1. cd stocks-app-new
```
```
2. npm install
```
``` 
3. npm start 
```
## Deployment

```
npm run deploy
```

## Built With

* [reactJs](https://reactjs.org/)
